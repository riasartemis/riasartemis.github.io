值初始化语法

C++ 中的值初始化语法有以下几种形式

  1. T()

  2. new T()

  3. Class:Class(...):member(){...}

  4. T object{}

  5. T{}

  6. new T{}

  7. Class::Class(...):member{}{...}

其中第4，5，6，7几种语法是从 C++11 之后开始的语法。需要额外注意的是，第四种语法用于命名对象（自动，静态，线程本地）的声明，并且它的初始化器由一对花括号组成。

在所有情况中，如果使用了空花括号对 { } 并且 T 是一个聚合类型，那么将会执行聚合初始化。

值初始化的效果（从C++11开始）

  1. 如果 T 是一个没有默认构造器的 Class 类型，或者是有一个用户提供的或者被删除的默认构造器的 Class 类型，那么将会执行默认初始化。

  2. 如果T是一个 Class 类型，并且它的默认构造器既不是用户提供的也不是被删除的（隐式定义的构造器或者被显示默认的默认构造器），那么就会对该对象执行零初始化（zero-initialized），但是如果 T 拥有 non-trivial 默认构造器，那么仍然会执行默认初始化

  3. 如果T是一个数组类型，那么每一个元素将会执行值初始化。

  4. 否则对象将会被零初始化（zero-initialized）。

上面的效果中，用户提供（user-provided）的函数指的是一个函数在它的第一次声明时，是用户声明的且没有被显示的删除或者默认。
